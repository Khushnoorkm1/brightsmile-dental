# 🦷 BrightSmile Dental — Next.js Website

A premium, production-ready dental practice website built with **Next.js 14**, **TypeScript**, and **Tailwind CSS**.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## ⚙️ Customize

All client details in one file: `src/lib/config.ts`

## 📄 Pages
- `/` Home
- `/services` Services
- `/about` About Us
- `/booking` Book Online
- `/contact` Contact

## 💬 Features
- WhatsApp Booking System
- Full SEO (Schema.org, Open Graph)
- Fully Responsive
- Booking Form with Time Slots
- Floating Action Bar

## 🌐 Deploy
```bash
npm install -g vercel && vercel
```
